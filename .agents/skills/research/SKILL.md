# /research

Research supports Gradient decisions; it does not smuggle private deployment
context into this repo.

## Source Priority

1. Repo sources: `docs/background.md`, `docs/decision-log.md`,
   `docs/architecture.md`, `docs/module-contracts.md`,
   `docs/source-dependencies.md`, `docs/experimentation-roadmap.md`.
2. Public official sources for modern tools, models, standards, and APIs.
3. Local reference repos only through public-safe summaries unless the target
   file is intended to be copied into Gradient.

## Procedure

1. State the decision the research will inform.
2. Check repo docs before web or external tools.
3. For current model/API/tool facts, verify from primary sources.
4. Summarize with citations and freshness.
5. Classify output as core, profile, adapter, policy, context, or private.

## Output

Return findings, sources, confidence, and the Gradient artifact likely to
change.
